# 4️⃣ 📦 GESTION DU ZIP

---

## 🧠 Règle fondamentale

```text
Le ZIP n’est PAS immédiatement disponible (surtout en National)
```

---

## 📍 URLs

### Régional (rapide)

```text
/webroot/json/zip_photos_{id}.zip
```

### National (lent ⚠️)

```text
/webroot/outils/telechargements/{id}
```

---

## ⚠️ Règle critique

```text
Un ZIP peut exister MAIS être incomplet
```

---

## ✅ Validation correcte

Un ZIP est prêt si :

```text
✔ taille > 0
✔ taille stable entre 2 checks
OU
✔ téléchargement réussi
```

---

## ❌ Interdit

```php
if ($size > 0) → OK ❌
```
